# Disconnect-WDServer

## SYNOPSIS
Closes the current session to Wikidata servers.

## SYNTAX

```
Disconnect-WDServer [<CommonParameters>]
```

## DESCRIPTION
Closes the current session to Wikidata servers wich was opened with Connect-WDServer.

## EXAMPLES

### Example 1
```
PS C:\> Disconnect-WDSertver
```

Closes the session.
